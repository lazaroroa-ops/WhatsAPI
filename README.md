# WhatsAPI Mail Service
A secure mail API with a terminal-based (TUI) frontend. 

## To install:
pip install -r requirements.txt

## If you are on Windows, curses isn't pre-installed in python, do:
pip install windows-curses

## To execute:
**First execute the backend:**
cd backend
py main.py

**Then the frontend:**
cd frontend
py main.py

**You can acces the interactive docs at "https://localhost:5000/apidocs/**
